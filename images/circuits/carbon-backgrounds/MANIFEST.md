# Circuit Carbon Background Images

This directory should contain carbon background images for all F1 circuits.

## Image Files Required

- `Great Britain carbon.png` - Silverstone
- `Belgium carbon.png` - Spa-Francorchamps
- `Italy carbon.png` - Monza
- `Monaco carbon.png` - Monaco
- `China carbon.png` - Shanghai
- `Japan carbon.png` - Suzuka
- `Singapore carbon.png` - Marina Bay
- `Australia carbon.png` - Albert Park
- `USA carbon.png` - Circuit of the Americas
- `Brazil carbon.png` - Interlagos
- `Azerbaijan carbon.png` - Baku
- `Spain carbon.png` - Catalunya
- `Hungary carbon.png` - Hungaroring
- `Austria carbon.png` - Red Bull Ring
- `Netherlands carbon.png` - Zandvoort
- `Abu Dhabi carbon.png` - Yas Marina
- `Saudi Arabia carbon.png` - Jeddah
- `Qatar carbon.png` - Losail
- `Bahrain carbon.png` - Bahrain
- `Las Vegas carbon.png` - Las Vegas
- `Miami carbon.png` - Miami
- `Emilia Romagna carbon.png` - Imola
- `Canada carbon.png` - Gilles Villeneuve
- `Mexico carbon.png` - Rodriguez

## Source URLs

All images can be downloaded from:
```
https://media.formula1.com/image/upload/f_auto,c_limit,w_1440,q_auto/f_auto/q_auto/content/dam/fom-website/2018-redesign-assets/Track%20icons%204x3/{circuitName}%20carbon
```

Replace `{circuitName}` with the circuit name from the list above (with spaces).
